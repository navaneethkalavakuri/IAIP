Project Description

This project is a Python-based Tic Tac Toe Game developed as part of the Intern Alpha Python Programming Internship.

The game provides a 3×3 grid where two players can play by taking turns marking X and O. The program checks for winning conditions after every move, detects draw situations, and provides an option to play the game again.

---

Objectives

- Create a 3×3 Tic Tac Toe board.
- Display the game board to players.
- Allow two players to play alternately.
- Check for a winner after each move.
- Detect draw situations.
- Provide a replay option.

Game Rules

1. The game is played on a 3×3 grid.
2. Player X always starts first.
3. Players take turns choosing positions from 1 to 9.
4. The first player to align three symbols horizontally, vertically, or diagonally wins.
5. If all positions are filled without a winner, the game ends in a draw.
